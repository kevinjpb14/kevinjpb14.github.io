## Publicar en GitHub Pages

### Opción 1: repositorio de usuario
1. Crea un repositorio con el nombre: `TUUSUARIO.github.io`
2. Sube estos archivos a la raíz del repo.
3. Haz commit y push.
4. Tu sitio quedará disponible en: `https://TUUSUARIO.github.io`

### Opción 2: repositorio de proyecto
1. Crea un repo, por ejemplo: `portfolio`
2. Sube estos archivos.
3. En GitHub entra a `Settings > Pages`
4. En `Build and deployment`, elige `Deploy from a branch`
5. Selecciona la rama `main` y la carpeta `/root`
6. Tu web quedará en: `https://TUUSUARIO.github.io/portfolio`
